🛡️ RCD Requirements — NZ (Clause 2.6)

AS/NZS 3000:2018 Clause 2.6. New Zealand requirements differ from Australia — see 2.6.2.2.3 and 2.6.3.3.

NZ Key Requirements:
• All RCDs must be Type A (2.6.2.2.3) — no Type AC for new installs
• Must interrupt all live conductors (active + neutral) (2.6.2.2.3)
• Residential: 30mA on all socket-outlet, lighting, and hand-held equipment circuits (2.6.3.3.1)
• Max 3 final subcircuits per RCD, min 2 RCDs in residential (2.6.2.4)
• Lighting distributed between RCDs if >1 installed (2.6.2.4)
• Range/cooktop socket can be exempt if marked and not for general use (2.6.3.3.1 Exception 2)
• School Yr8 & below: 10mA RCD on socket-outlets (2.6.3.3.2(b))
• Children's rides, vending machines: 30mA RCD (2.6.3.3.2(d))
